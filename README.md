# 크몽 카테고리 추천 시스템

이 프로그램은 기업 정보를 바탕으로 크몽 Biz 구매자에게 적합한 카테고리를 추천하는 시스템입니다.

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. `.env` 파일 설정:
- `.env` 파일을 열고 `OPENAI_API_KEY`에 본인의 OpenAI API 키를 입력하세요.

## 실행 방법

```bash
python main.py
```

## 입력 형식

프로그램 실행 시 다음과 같은 형식으로 입력하세요:
```
기업명,사업자번호
```

예시:
```
크몽,123-45-67890
```

## 출력 결과

프로그램은 입력된 기업 정보를 바탕으로 다음을 출력합니다:
- 기업 정보
- 추천 카테고리 목록
- 각 카테고리 추천 근거
